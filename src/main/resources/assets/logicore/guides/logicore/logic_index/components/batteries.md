# Batteries

## Batteries

Once you start generating power, you can use the batteries to store it. Batteries come in different sizes, each with
different capacity and transfer rates. All batteries can be modified via the config file to change their capacity and
transfer rate.

The small battery is the first of it's kind with a small capacity and transfer rate.
