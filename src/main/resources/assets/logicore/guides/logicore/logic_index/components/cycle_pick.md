# Cycle Pickaxe

## Cycle Pickaxe

<ItemImage id="logicore:cycle_pick" />

The Cycle Pickaxe is a high-performance mining tool designed for efficiency. It features exceptional durability (3000)
and mining speed (12.0), making it superior to most conventional tools.

Its true potential is unlocked when enchanted with the [Recycler](logicore:architecture/recycler.md) enchantment.
