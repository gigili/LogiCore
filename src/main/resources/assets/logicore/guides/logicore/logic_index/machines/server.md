# Server

## Server

<ItemImage id="logicore:server" />

A portable computing unit that houses up to 9 [Processor Units](../components/processor_unit.md) in a hot-swappable
chassis.

Servers are the primary processing modules for the [Server Rack](./server_rack.md), allowing dense CPU configurations in
a compact form.

## Configuration

Right-click the Server in your hand to open its configuration panel. Insert Processor Units (CPUs) into the 9 available
slots to populate the server.

The Server's tooltip will display a breakdown of installed CPUs by tier and the total [Cycle](../architecture/cycles.md)
generation rate.

## Usage

Once populated with CPUs, insert the Server into a [Server Rack](./server_rack.md) by right-clicking the rack while
holding the Server.

Servers can be removed via the Server Rack's GUI. Each Server accepts all three CPU tiers: Basic, Advance, and Ultimate.

A hot-swappable blade chassis for high-density processor deployment.
