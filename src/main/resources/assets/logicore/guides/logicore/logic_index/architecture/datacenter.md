# Datacenter

The Datacenter is a massive sealed structure used to house Server Racks and regulate their temperature.

It is a hollow cuboid.

- Min Size: 7x7x7
- Max Size: 32x32x32

Unless the configs were changed. The edges must be *Frames*. The faces must be *Walls*.
