# Stack upgrade

## Stack upgrade

<ItemImage id="logicore:stack_upgrade" />

This upgrade allows you to change the amount of items a [Compiler](../machines/compiler.md) produces. By default each
stack upgrade gives you 4 extra items. With a total of 16 upgrades, your compiler will produce 64 items.
