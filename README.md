# LogiCore

# For devs

You can add this mod to your project by adding this:

```
maven {
    name = "Gac Repository"
    url = uri("https://maven.igorilic.dev/releases")
}
```

And then adding this as a dependency:

```
implementation "dev.gacbl:logicore:${lc_version}"
```
